# 반복문

## for문
```java
// A부터 B까지 증가
for(int i=A; i<=B; i++)

// A부터 B까지 C씩 증가
for(int i=A; i<=B; i+=C)

// A부터 B까지 감소
for(int i=A; i>=B; i--)
```
```java
// 1, 2, 3, 4, 5
for(int i=1; i<=5; i++)

// 5, 10, 15, 20
for(int i=5; i<=20; i+=5)

// 4, 7, 10, 13, 16, 19
for(int i=4; i<=19; i+=3)

// 10, 8, 6, 4
for(int i=10; i>=4; i-=2)

// 3, 1, -1
for(int i=3; i>=-1; i-=2)
```
